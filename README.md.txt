# Student Performance Data Cleaning and Visualization

## Objective
To clean, analyze, and visualize the Student Performance Dataset using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Dataset
StudentsPerformance.csv

## Tasks Performed
1. Data Loading
2. Data Inspection
3. Missing Value Analysis
4. Duplicate Record Check
5. Statistical Analysis
6. Data Visualization

## Visualizations
- Math Score Distribution Histogram
- Gender Distribution Count Plot
- Reading vs Writing Score Scatter Plot
- Correlation Heatmap

## Results
- Dataset contains 1000 records and 8 columns.
- No missing values found.
- No duplicate records found.
- Strong positive correlation between reading and writing scores.

## Conclusion
The dataset was successfully analyzed and visualized to understand student performance patterns.
